HTML lists

A author can create list of different items using HTML lists

Example

# Lists of Students
# Lists of Players
# Lists of colors
# Lists of fruits

Different ways to create a list

# <ul> create a unordered list without a specific order
# <ol> create a ordered list with proper numbering ordering
# <dl> create a defined list like dictionary

unordered lists

<ul> <ul/> is used to create unordered list
<li> <li/> is used to define the items inside lists
  • Red
  • Black
  • Green
  • Yellow

ordered lists

<ol> <ol/> is used to create unordered list
<li> <li/> is used to define the items inside lists
  1. Red
  2. Black
  3. Green
  4. Yellow
  1. Red
  2. Black
  3. Green
  4. Yellow
